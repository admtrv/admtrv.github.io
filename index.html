typecrab

A lightweight and customizable typing test built in Rust, featuring both CLI and Web UI. Designed as a fast, cross-platform alternative to existing tools, focused on minimalism, performance, and flexible user settings.

Physical Simulations

A collection of standalone physical simulations developed in C++ using OpenGL/GLUT. Created as part of the Physical Basics of Computer Games course to explore fundamental mechanics and visualize physics problems interactively.
