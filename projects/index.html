typecrab

A lightweight and customizable typing test built in Rust, featuring both CLI and Web UI. Designed as a fast, cross-platform alternative to existing tools, focused on minimalism, performance, and flexible user settings.

Physical Simulations

A collection of standalone physical simulations developed in C++ using OpenGL/GLUT. Created as part of the Physical Basics of Computer Games course to explore fundamental mechanics and visualize physics problems interactively.

RayBoy

Raycasting visualization adapted for the Arduboy platform. Uses the DDA algorithm for efficient rendering on limited hardware. Based on the original CLI Raycaster project.

Playroom Graphic Demo

A 3D interactive graphics demo made in C++ with OpenGL. It simulates a playroom with animated lighting, post-processing, procedural outdoors, and interactive physics-based games like billiards and darts.

BigNumbers

Header-only C++ library for arbitrary-precision integers and rationals. Includes full operator support, JSON expression evaluation, and optional features like square root and primality testing.

HelloPeer!

A p2p application written in C++ for message exchange between two nodes on a local Ethernet network. It uses a custom UDP-based protocol with mechanisms for reliable data transfer between devices acting as both senders and receivers.

Data Structures and Algorithms

A collection of complex data structures and algorithms implemented in pure C. Originally created for the university course, this project is now publicly available for learning.

CLI Raycaster

Command-line raycasting visualization written in pure C using the curses library. Created for educational purposes to explore the classic technique of simulating fake 3D used in early games like Wolfenstein 3D.
