HelloPeer!

A p2p application written in C++ for message exchange between two nodes on a local Ethernet network. It uses a custom UDP-based protocol with mechanisms for reliable data transfer between devices acting as both senders and receivers.

Playroom Graphic Demo

A 3D interactive graphics demo made in C++ with OpenGL. It simulates a playroom with animated lighting, post-processing, procedural outdoors, and interactive physics-based games like billiards and darts.

CLI Raycaster

Command-line raycasting visualization written in pure C using the curses library. Created for educational purposes to explore the classic technique of simulating fake 3D used in early games like Wolfenstein 3D.
